ITK API and Reference Implementation mini-site

Project Modules

This project has declared the following modules:

NameDescription
ITK APICFH ITK Java Application Programming Interface (API). This is a set of Java interfaces that provide a suitable abstraction layer for clinical applications. The API abstracts the application from the mechanics of the ITK specification for transporting and distributing messages.
ITK Java Reference ImplementationCFH ITK Java reference implementation of the ITK Java API. The purpose of the reference implementation is to provide a simple yet functional implementation of the ITK Java APIs for use as education and reference as well some potential real world usage in lightweight ITK compliant applications.
ITK Samples projectCFH ITK samples project showing how the ITK API and reference implementation can be used to fulfil some common use-cases
